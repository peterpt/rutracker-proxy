Tuning Deluge

1. Open Deluge and go to: Edit -> Options

2. Click the "Proxy" tab, set the values ​​as in the screenshot and click "OK"

Configuring qBittorrent

1. Open qBittorrent and go to: Tools -> Options ...

2. Click the "Connection" tab, set the values ​​as in the screenshot and click "OK"

Configuring µTorrent

1. Open uTorrent and go to: Settings -> Application Settings

2. Click the "Connection" tab, set the values ​​as in the screenshot and click "OK"

Configuring Vuze

1. Open Vuze and go to: Tools -> Settings ...

2. Click the "Mode" tab and select "Professional" as in the screenshot.

3. Click the Connection -> Proxy tab, set the values ​​as in the screenshot, click "Save" and reload the Vuze
